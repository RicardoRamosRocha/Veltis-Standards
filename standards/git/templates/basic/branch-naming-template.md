# Template de Nome de Branch

## Formato

```text
tipo/descricao-curta
```

## Tipos

- `feat/`: funcionalidade.
- `fix/`: correção.
- `docs/`: documentação.
- `chore/`: manutenção.
- `refactor/`: refatoração.
- `test/`: testes.
- `release/`: preparação de release.
- `hotfix/`: correção urgente.

## Exemplos

```text
feat/convite-usuarios
fix/corrige-validacao-email
docs/padrao-git
chore/atualiza-dependencias
release/1.2.0
hotfix/1.2.1-corrige-login
```

## Checklist

- [ ] Nome em minúsculas.
- [ ] Sem acentos.
- [ ] Descrição curta e objetiva.
- [ ] Tipo compatível com a mudança.
- [ ] Escopo adequado para um pull request revisável.
