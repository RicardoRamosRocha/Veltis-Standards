## Histórias

### US-001 - Abrir chamado

Como cliente, quero abrir um chamado com descrição e categoria, para registrar uma solicitação de atendimento.

- Prioridade: Alta
- Status: Planejado
- Versão prevista: 1.0

Critérios de aceite:

- O formulário exige categoria e descrição.
- O chamado recebe número identificador.
- O usuário visualiza confirmação após envio.

### US-002 - Acompanhar status

Como cliente, quero acompanhar o status do chamado, para entender o andamento da solicitação.

- Prioridade: Alta
- Status: Planejado
- Versão prevista: 1.0

Critérios de aceite:

- A lista mostra chamados do usuário autenticado.
- Cada chamado exibe status, data e prioridade.
