## [1.0.0] - 2026-06-26

### Adicionado

- Guias oficiais em `docs/` para arquitetura geral, camadas, Clean Architecture, DDD, injeção de dependência, Repository Pattern, Service Layer, CQRS, monólito modular, estrutura de projeto, nomenclatura e decisões arquiteturais.
- Templates básicos para documento de arquitetura, ADR, descrição de camada e estrutura de projeto.
- Templates avançados para Clean Architecture, monólito modular, SaaS, arquitetura corporativa e revisão arquitetural.
- Exemplos básicos de arquitetura em camadas, Clean Architecture, Repository Pattern, injeção de dependência e monólito modular.
- Exemplos de projetos reais para Veltis AI Platform, Veltor ERP e Renova.
- Estudos de caso para Veltis AI Platform, Veltor ERP e Renova.
- ADRs sobre arquitetura em camadas, Clean Architecture, monólito modular, camada de domínio e camada de infraestrutura.
- Release notes `releases/v1.0.md`.
